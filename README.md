# Runway 队列助手

一个 Chrome 扩展插件，用于自动将提示词队列批量提交到 Runway ML 生成。支持 Runway Unlimited 模式的 2 个并发槽位，适合动画/影视工作室批量生产分镜视频。

## 功能特性

- **队列管理**：批量添加提示词任务
- **批量导入**：支持分隔符批量粘贴，一次导入 30-60 个任务
- **并发提交**：自动利用 Runway Unlimited 的 2 个并发槽位，前一个提交后立即提交下一个
- **参考图绑定**：自动识别 `@参考图片`，逐字输入后等待 1 秒再回车，确保 Runway 正确绑定
- **状态机驱动**：pending → running → completed 三状态，基于 Generate 按钮状态判断槽位
- **可调参数**：检查间隔、延迟等均可配置
- **暂停支持**：运行中点击暂停，可在逐字输入过程中中断
- **测试模式**：只输入提示词不提交，方便验证 @参考图 绑定效果

## 安装步骤

### 1. 准备图标（可选）

插件目录已包含默认图标（`icons/` 下），可直接使用。如需自定义：

**方式 A：在线转换**
1. 打开 https://cloudconvert.com/svg-to-png
2. 上传 `icons/icon.svg`
3. 输出 128x128 PNG
4. 替换 `icons/icon128.png`

**方式 B：macOS 预览应用**
1. 用 Preview 打开 `icons/icon.svg`
2. 导出为 PNG，替换原文件

### 2. 加载插件到 Chrome

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `runway-queue-extension` 文件夹
5. 插件图标会出现在 Chrome 工具栏

### 3. 验证安装

1. 打开 Runway 网站并登录（https://app.runwayml.com）
2. 按 F12 打开开发者工具 → Console
3. 应该能看到 `[Runway Queue] 内容脚本已加载`
4. 点击 Chrome 工具栏的插件图标，确认弹窗正常显示

## 使用手册

### 基本流程

1. 打开 Runway 网站（https://app.runwayml.com）并登录
2. 点击 Chrome 工具栏的插件图标，打开弹窗面板
3. 添加提示词到队列（单个添加或批量导入）
4. 点击「开始」按钮，插件自动逐个提交
5. 可随时点击「停止」暂停

**单个添加：**
- 在文本框中输入提示词（支持 `@文件名` 语法引用图片）
- 点击「添加到队列」或按 `Ctrl+Enter` / `Cmd+Enter`

**批量导入：**
- 点击「批量导入」打开批量导入弹窗
- 将多个提示词用分隔符（默认 `---`）隔开，粘贴到大文本框
- 自动解析并显示预览
- 确认无误后点击「确认添加」全部加入队列

### 批量导入详解

适用于每天 30-60 个任务的场景。在 Excel 中管理分镜提示词时：

**Excel 操作步骤：**
1. 确保提示词在单独一列（如 B 列）
2. 在 C 列添加辅助公式：`=B1 & CHAR(10) & "---"`
3. 下拉填充整列
4. 复制 C 列，粘贴到批量导入文本框中
5. 自动解析，预览确认后添加

**分隔符：** 默认为 `---`，可自定义修改。解析结果实时显示。

**注意事项：**
- 每个提示词最大 3000 字，一次可导入 30-60 个
- 空任务会被自动过滤
- 批量添加的任务 ID 唯一，互不冲突

### 提示词格式

**普通提示词：**
```
A cute cartoon crayfish in a magical forest
```

**带参考图片的提示词：**
```
洪桂萍 @hong.png
唐思静 @tang.png
陈俊宏 @chen.png
【核心剧情】
清晨六点，主卧的遮光帘被拉开...
```

输入要点：
- `@参考图` 会被自动识别并逐字输入
- 每个 `@参考图` 输入完成后等待 1 秒，然后按回车绑定
- 普通文本正常速度输入
- 参考图名称中不要有空格

### 参数设置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 检查间隔 | 150 秒 | 轮询检查槽位和完成状态的间隔 |
| 成功后延迟 | 5 秒 | 每次任务提交后的固定等待 |
| 随机延迟 | 5 秒 | 额外随机延迟范围（0-N 秒） |
| 默认时长 | 15s | Runway 视频时长设置（也可不设置） |
| 测试模式 | 关闭 | 开启后只输入不提交，用于验证 |

### 并发机制

Runway Unlimited 模式有 **2 个并发生成槽位**。插件通过检查 Generate 按钮状态判断槽位：

1. 有槽位且有待提交任务 → 开始处理（标记 running）
2. 逐字输入提示词 + 点击 Generate → 标记 completed
3. Generate 按钮禁用 → 槽位满，等待
4. Generate 按钮恢复 → 有空槽位 → 提交下一个
5. 全部任务 completed/failed → 自动停止

### 任务状态说明

| 状态 | 图标 | 含义 |
|------|------|------|
| 等待中 | ○ | 尚未提交到 Runway |
| 处理中 | ⟳ | 正在输入提示词或点击 Generate 按钮 |
| 已完成 | ✓ | 已成功点击 Generate 按钮 |
| 失败 | ✗ | 提交过程中出现错误 |

## 常见问题

### 参考图没有绑定成功？
- 确认提示词中 `@参考图` 格式正确（不要有空格）
- 开启测试模式，手动检查输入框中的 @参考图是否被正确识别
- 检查控制台日志中是否有"参考图已回车绑定"的输出

### 任务一直不推进？
- 查看控制台日志确认当前 running 任务数和 Generate 按钮状态
- 如果 Runway 页面出现问题，刷新页面后重新开始队列
- 确认 Runway 页面保持在当前标签页（不要切换到其他标签页）

### 剪贴板输入失败？
- 插件会自动回退到逐字输入方式，不影响使用
- 控制台中会显示"回退到逐字符输入"

### 点击暂停没有立即生效？
- 如果正在逐字输入过程中，插件会在几个字符内检测到暂停并中断
- 如果已点击生成按钮，当前任务会完成提交后才停止

## 调试

### 查看运行日志

1. 打开 Runway 页面
2. 按 F12 打开开发者工具 → Console
3. 所有日志以 `[Runway Queue]` 为前缀

[Runway Queue] 输入参考图: @hong
[Runway Queue] 等待 1 秒后回车绑定参考图...
[Runway Queue] 参考图已回车绑定: @hong

### 手动检查状态

在 Runway 页面的 Console 中执行：

```javascript
// 检查内容脚本是否加载
window.__runwayQueueLoaded

// 查看队列状态
chrome.storage.local.get(['queue', 'isRunning', 'settings']).then(console.log)

// 检查输入框
document.querySelector('[aria-label="Prompt"]')
```

## 风险提示

- Runway 的服务条款可能禁止自动化操作，使用风险自负
- 建议适当增加检查间隔和随机延迟，降低检测风险
- 首次使用建议开启测试模式验证参考图绑定
- 建议先测试 1-2 条任务，观察是否正常
- 生成超时不宜设置过短，否则可能误判任务完成
